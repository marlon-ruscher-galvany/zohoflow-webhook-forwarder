# zohoflow-webhook-forwarder
A small middleware enabling meta to use zoho flow webhook (because ZOHO Flow does not support webhook responses not standard webhook verifications).

### Usage
`https://zohoflow-webhook-forwarder.vercel.app/api/hook?target=https://flow.zoho.eu/2010....5/flow/webhook/incoming?zapikey=1001.a...`

And set the verification code to
`zuj7erv!ngy1xtf2PKZ`
